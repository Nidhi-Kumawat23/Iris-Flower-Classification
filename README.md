🌸 Iris Flower Classification using Logistic Regression

📌 Project Overview

This project implements a Machine Learning model to classify iris flowers into different species based on their features.

The classification is performed using the Logistic Regression algorithm, achieving high accuracy on the dataset.

🌼 Dataset Information

The project uses the famous Iris dataset, which contains:

150 samples

3 classes:

Setosa

Versicolor

Virginica

Features:

Sepal Length

Sepal Width

Petal Length

Petal Width

🛠 Technologies Used

Python

pandas

NumPy

Seaborn

Matplotlib

Scikit-learn

📂 Project Workflow

1️⃣ Import Libraries

Imported required Python libraries for data handling, visualization, and machine learning.

2️⃣ Load Dataset

Loaded the Iris dataset and converted it into a pandas DataFrame for better understanding and manipulation.

3️⃣ Data Visualization

Used Seaborn pairplot to visualize relationships between features.

Helped understand how different classes are distributed.

4️⃣ Train-Test Split

Split dataset into training and testing data using:

test_size = 0.2

random_state = 42

5️⃣ Model Training

Applied Logistic Regression model using Scikit-learn

Trained the model on training data.

6️⃣ Predictions

Predicted flower classes using the trained model on test data.

7️⃣ Model Evaluation

Accuracy Score:

1.0 (100% accuracy)

Confusion Matrix:

Used to evaluate the performance of the classification model.

📊 Results

The model achieved 100% accuracy, indicating excellent performance on the dataset.
